Get All Tourist Attractions

This method retrieves a list of all tourist attractions.

  1. Make a GET request to the /attractions endpoint.
  2. Returns a JSON array with the list of attractions.
  3. Responds with HTTP Status 200 (OK).

Get Tourist Attraction by Name

This method retrieves information for a specific tourist attraction by name.

  • Make a GET request to /attractions/{name}.
  • Returns a JSON object with the attraction details.
  • Responds with HTTP Status 200 (OK).

Add New Tourist Attraction

This method allows the addition of a new tourist attraction.

  • Make a POST request to /attractions/add with the attraction data in the request body.
  • Returns the created attraction object in the response.
  • Responds with HTTP Status 201 (Created).

Update Tourist Attraction

This method updates an existing tourist attraction based on its name.

  1. Make a POST request to /attractions/update with the attraction's name and updated details in the request body.
  2. Returns the updated attraction object in the response.
  3. Responds with HTTP Status 200 (OK).

Delete Tourist Attraction

This method deletes a tourist attraction by its name.

  • Make a POST request to /attractions/delete/{name}.
  • Returns the deleted attraction object.
  • Responds with HTTP Status 200 (OK).
